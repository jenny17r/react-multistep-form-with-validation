# React Multi-Step Form

A multi-step form application that guides users through different steps to complete a form. This project demonstrates step-based navigation and form handling in React.

---

## Features

* Step-by-step form navigation
* Next & Back functionality
* Form data persistence across steps
* Clean UI design

---

## Tech Stack

* React (Vite)
* JavaScript
* Tailwind CSS

---

## Concepts Covered

* useState for step control
* Conditional rendering
* Lifting state up
* Form handling
* Component separation

---

## Steps

1. Personal Information
2. Account Details
3. Confirmation

---

## How to Run

```bash
npm install
npm run dev
```
